Build Tools
Apache ANT
Probably the easiest and longest used java build tool
Apache Maven
Its suppose to improve what ANT could not.
Apache Ivy
Not a build tool, but used by build tools to manage dependencies.
Apache Buildr
A ruby build tol that acts like ant.

Groovy's version of an ant java build tool
